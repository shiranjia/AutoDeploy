#AutoDeploy
an auto deploy tool  自动部署工具

任意添加本地命令
任意添加远程命令
任意上传文件或目录</br>
######获取方式：$ go get github.com/shiranjia/AutoDeploy</br>
######编译：AutoDeploy/src$ go install AutoDeploy</br>
######../bin目录下生成 ./AutoDeploy 可执行文件，直接运行，浏览器访问127.0.0.1即可</br>
